# Json_csv
```
json파일의 내용을 요구된 format에 맞춰 csv파일로 생성
```
## 🐰 역할
- 기존의 값 df에 저장 (변수명 : dataFrame)
- 새로 추가된 key 생성 후 기존 값 저장한 df에 추가해서 csv 생성 (변수명 : dataFrame_add)
<br>
👉 폴더 내 모든 json읽고 요구된 format에 맞춰 csv형태로 내용을 옮기는 코드
